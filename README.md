## Quick Start

### Prerequisites
- Node.js 16.x or later
- npm, yarn, or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/onesimusaz/azraokproject.template.nuxt3.git

# Navigate to the project directory
cd azraokproject.template.nuxt3

# Install dependencies
npm install
# or
yarn install
# or
pnpm install
```

### Development

Start the development server on http://localhost:3000

```bash
npm run dev
```

### Production

Build the application for production:

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
```

## Customization

### Styling
- Main styles: `assets/scss/style.scss`
- Variables: `assets/scss/_variables.scss`
- Components: `assets/scss/component/`

### Components
- Reusable components in `/components` directory
- Pages in `/pages` directory with auto-routing
